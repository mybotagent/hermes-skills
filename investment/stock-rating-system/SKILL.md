# S+~F Stock Rating System

**관련 스킬**: `fair-value-portfolio` (정량 밸류에이션 데이터 제공)

## 개요
종목을 8개 카테고리로 평가하여 S+(최고) ~ F(최하) 등급을 부여하는 정성·정량 통합 프레임워크.

## ⚠️ 사용자 선호: 평가 방식 (반드시 준수)

이 스킬을 사용할 때 다음 규칙을 **절대 위반하지 말 것**:

1. **코드 생성 금지**: 사용자가 "평가해줘"라고 요청하면 절대 Python 코드/스크립트를 만들지 말 것. "코드를 만들어볼까요?", "코드부터 짜겠습니다" 같은 접근도 금지. **그냥 직접 평가 결과를 마크다운 테이블로 출력할 것.**
2. **하드코딩 금지**: 평가 기준과 방법론만 정의할 것. 특정 종목(ticker, 이름)을 스킬 본문에 하드코딩하지 말 것. 종목 데이터는 스킬 외부(실행 시점 수집 or 사용자 입력)에서 가져옴.
3. **형식**: 마크다운 테이블로 S+~F 등급을 표시하고, 각 종목에 1~2줄 정성 코멘트. 불필요한 방법론 설명 생략.
4. **등급 체계**: S+~F 사용. 각 항목별 등급 + 종합 점수 함께 표시.
5. **적정PER 패널티 엄격 적용**: 사용자 핵심 원칙 = "적정 PER 내에서만 투자". AbsVal이 D+ 이하면 Composite을 B로 제한하고, 매수 추천에서 제외 또는 별도 위험 표기를 필수로 할 것.

**이유**: 사용자는 직접 코드를 실행/검증하지 않음. 평가 결과 자체에 집중하며, 방법론 설명이나 코드 생성은 불필요하게 느낌.

## Phase 3 포트폴리오 비중과의 연결 (2026-06-08)

**중요**: Phase 3 포트폴리오 비중 산정 시 이 스킬의 8항목 Composite 점수를 **직접 사용하지 않음**.

대신, 아래의 **Economic Moat 항목의 기준**을 LLM이 참고하여 1~10점으로 단순화 추정:
- 9~10: 독점적 지위 (ROE 30%↑ + 영업이익률 30%↑) — NVDA, MSFT, GOOGL 등
- 7~8: 강한 시장지위 (ROE 15~30% + 이익률 15~30%)
- 5~6: 차별화 보통 (ROE 10~15% + 이익률 5~15%)
- 3~4: 차별화 약함, 경쟁 심함 (ROE 5~10%)
- 1~2: 진입장벽 없음 (ROE 0~5% or 적자)

**최종 수식**: `가중값 = max(0, t1_gap) × moat_점수` — 오직 gap + moat 두 요소만 사용.
PER상태/성장성/현금흐름 등 다른 요소는 포트폴리오 비중에서 **반영 금지**.

이 8항목 프레임워크는 사용자가 전체 등급 평가를 요청할 때 (예: "S+~F 평가해줘") 별도로 사용.

### 1️⃣ Economic Moat (경제적 해자) — 가중치 35%
경쟁사가 넘기 어려운 진입장벽을 측정.

| 등급 | 기준 |
|:----:|------|
| **S** | ROE 30%↑ + 영업이익률 30%↑ + 시총 1000억↑ + 독점적 시장지위 |
| **A+~A** | ROE 20~30% + 이익률 15~30% + 시총 200~1000억 + 강한 브랜드 |
| **B+~B** | ROE 10~20% + 이익률 5~15% + 시총 30~200억 + 차별화 약함 |
| **C+~C** | ROE 5~10% + 이익률 0~5% + 시총 10~30억 + 경쟁 심함 |
| **D~F** | ROE 0~5% or 적자 + 시총 10억 미만 + 진입장벽 없음 |

**판단 포인트**: 특허, 네트워크 효과, 전환 비용, 브랜드 파워, 규제 라이선스

### 2️⃣ Growth (성장성) — 가중치 25%
매출·이익·FPER의 성장 궤적.

| 등급 | 기준 |
|:----:|------|
| **S+** | 매출성장률 50%↑ + EPS성장 50%↑ + FPER가 PER의 50% 미만 (급성장) |
| **S** | 매출 30~50% + EPS 30~50% + FPER/PER 50~70% |
| **A+~A** | 매출 15~30% + EPS 15~30% + 건강한 성장 |
| **B+~B** | 매출 5~15% + EPS 5~15% + 안정적 성장 |
| **C+~C** | 매출 0~5% + EPS 정체 |
| **D~F** | 매출 감소 + EPS 하락 or 적자 지속 |

**판단 포인트**: TAM(총가용시장) 크기, 시장점유율 추세, 신규 사업 모멘텀

### 3️⃣ Cash Flow (현금 창출력) — 가중치 20%
이익의 질과 현금 전환 능력.

| 등급 | 기준 |
|:----:|------|
| **S** | P/FCF 15↓ + 영업이익률 30%↑ + 부채비율 20%↓ |
| **A+~A** | P/FCF 15~25 + 이익률 15~30% + 부채 20~50% |
| **B+~B** | P/FCF 25~40 + 이익률 5~15% + 부채 50~100% |
| **C+~C** | P/FCF 40~60 + 이익률 0~5% + 부채 100~200% |
| **D~F** | P/FCF 60↑ + 적자 or 부채 200%↑ |

**판단 포인트**: FCF Yield, 감가상각 대비 Capex, 운전자본 효율

### 4️⃣ Absolute Valuation (절대 밸류에이션) — 강력 패널티 적용
**fair-value-portfolio 스킬의 Orbit v2 T0/T1 이중 괴리율 기준**. 가치투자 원칙 엄격 적용.

**T0 괴리율 기준** (현재 적정가 vs 현재가):
| 등급 | Orbit T0 괴리율 | 의미 |
|:----:|:--------------:|------|
| **S+~S** | -30% 이하 (저평가) | 적정가 대비 30%+ 저렴 |
| **A+~A** | -30% ~ -10% | 양호한 밸류에이션 |
| **B+~B** | -10% ~ +10% | 적정 범위 내 |
| **C+~C** | +10% ~ +30% | 약간 고평가 |
| **D+~D** | +30% ~ +60% | 상당한 고평가 ← **Composite B로 제한** |
| **F** | +60%↑ or P/FCF 100↑ | 심각한 고평가 ← **Composite C로 제한** |

**참고**: Orbit T0 = EPS_T0 × 적정PER × W + BPS_T0 × (ROE/9) × (1-W)
- W (PER:PBR 혼합비): clip(0.4 + growth_rate×0.4, 0.4, 0.8)
- 적정PER: 섹터 기준 + 성장/부채/ROE 프리미엄 (clip 5~35)
- 자세한 산식: `skill_view("fair-value-portfolio")` → 분석 방법론 참조

**T1 괴리율 보조 참고** (1년 후 Forward EPS 기반):
- T1 괴리율이 +20%↑ → 1년 후 기대 수익률 양호 (성장성 반영)
- 단, T1은 analyst FPER 기반이므로 25~35% 과대낙관 가능성 명시
- T1 괴리율이 T0보다 현저히 높으면 → FPER가 낮음 = 실적 급증 기대

### 5️⃣ Relative Valuation (상대 밸류에이션)
동종 업계 내 위치.

| 등급 | 기준 |
|:----:|------|
| **S** | 업계 PER 하위 20% (가장 저렴) |
| **A** | 하위 20~40% |
| **B** | 중위 40~60% |
| **C** | 상위 60~80% |
| **D~F** | 상위 20% (업계 내 가장 비쌈) |

### 6️⃣ Geopolitical Risk (지정학적 리스크)
무역 분쟁, 규제, 지역 리스크.

| 등급 | 기준 |
|:----:|------|
| **S+~A** | 리스크 거의 없음 (미국 헬스케어·소비재) |
| **B+~B** | 제한적 노출 |
| **C+~C** | 일부 노출 (중국 매출 10~20%) |
| **D+~D** | 상당한 리스크 (대만·반도체·중국 매출 30%↑) |
| **F** | 핵심 리스크 (反독점 소송·직접 수출통제 대상·NK 리스크) |

### 7️⃣ Competitor Risk (경쟁 리스크)
산업 내 경쟁 강도와 포지션.

| 등급 | 기준 |
|:----:|------|
| **S** | 독점 or 과점 (MS·구글 검색·TSMC 파운드리) |
| **A+~A** | 확실한 선두 + 높은 전환비용 |
| **B+~B** | 경쟁 우위 있으나 추격자 존재 |
| **C+~C** | 과당 경쟁, 차별화 약함 |
| **D~F** | 시장지위 하락 중 or 상품화 위험 |

### 8️⃣ AI Innovation (AI 혁신)
AI 시대의 포지셔닝: Disruptor인가, Disrupted인가.

| 등급 | 분류 | 기준 |
|:----:|:----:|------|
| **S+** | 🚀 **Core Disruptor** | AI 반도체·인프라·플랫폼 독점 (NVDA·MSFT·GOOGL·AVGO) |
| **S** | 🚀 **Disruptor** | AI 혁신 주도, 생태계 장악 (TSM·AMD) |
| **A+~A** | 🤖 **Strong Beneficiary** | AI 수요 폭발 직접 수혜 (MU·SK하이닉스·SNDK·LLY) |
| **B+~B** | 🤖 **Beneficiary** | AI 간접 수혜 or 기술 적용 중 (AAPL·DELL·CLS) |
| **C+~C** | ⚖️ **Neutral** | AI 영향 제한적 (현대차·삼성전기) |
| **D~F** | ⚠️ **At Risk** | AI가 비즈니스 모델 직접 위협 |

## 종합 점수 산정

```python
Composite = Moat_Score × 0.35 + Growth_Score × 0.25
          + CF_Score × 0.20 + AbsVal_Score × 0.20

# 패널티: AbsVal이 D+ 이하 → Composite 7.0(B) 상한
# 패널티: AbsVal이 F → Composite 6.0(C) 상한
```

## 정렬 기준 (순위 결정)
1. **Moat** (35%) — 해자가 가장 중요
2. **Growth** (25%) — 그 다음 성장성
3. **Cash Flow** (20%) — 현금 창출력
4. **AbsVal** (20%) — 밸류에이션

## 최종 필터 (Top Picks 선출 조건)
- **조건 A**: Moat·Growth·Cash Flow 모두 A+(8.5) 이상 → ⭐ Top Tier
- **조건 B**: 조건 A + AbsVal C+ 이상 → 최종 Buy List
- **조건 C**: Composite 8.0(A) 이상 → Top 10 List
- **AbsVal D+ 이하**: Valuation 경고 플래그, Composite 상한 제한

## 주의사항
- 이 프레임워크는 **출발점**이지 결론이 아님
- 정량 수치(yfinance, 재무제표) + 정성 판단(산업 동향, 경영진, 경쟁 구도)의 결합 필요
- GeoRisk와 AI Innovation은 특히 주관적 판단이 많이 개입되므로 지속적인 업데이트 필수
- 지정학적 리스크는 뉴스 플로우에 따라 급변할 수 있음
